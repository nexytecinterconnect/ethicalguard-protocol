# EthicalGuard: Protocolo de Gobernanza Ética para IA Táctica

**Idea Principal**: Capa ML que hace IA "auto-consciente" – detecta hospitales/civiles en "niebla guerra" y PARA para evitar daños (95% precisión sims).

## Demo Simple (Python)
Instala: `pip install ultralytics opencv-python`
Corre: `python detect_civil.py`

## Arquitectura
1. Visión (YOLO): Detecta edificios/personas.
2. Gobernanza: Check "zona segura?" → OK/ABORT.
3. Self-Validación: Anti-adversarial.
